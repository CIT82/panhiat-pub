I chose Navs and Tabs because navigation components are essential in a webpage, as they link to different pages within the website or app. Bootstrap makes this simple and easy to implement using an unordered list followed by list items.

I opted for a basic nav that is both flexible and centered, achieved by using the nav justify-content-center class. This navigation setup is versatile and can be easily adapted to different alignments such as vertical, horizontal, left-aligned.
